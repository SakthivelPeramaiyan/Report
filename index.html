How to Install

This one is pretty easy. To install Screenbot, simply unzip the file you downloaded into your main directory. Run Install file.

How to Use

Just open the application where you want to take screenshot and to document it, simply press CTRL+Shift+A to take screenshots, you will hear a notification sound to confirm that screenshot is taken. use Ctrl+Shift+A to take as many times you want. once you finished and would like to document it, press CTRL+Shift+W to generate a word document and wait for some time. Screenbot will create a formated nice word document with our Screenshots in one go.
